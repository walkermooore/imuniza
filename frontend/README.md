## 🌐 Portal do Usuário
Interface moderna desenvolvida para profissionais de saúde, com foco em usabilidade mobile-first e agilidade operacional.

---

## 🚀 Diferenciais UI/UX
- **Feedback Imediato**: Alertas visuais de segurança para ações críticas.
- **Dark Mode Ready**: Suporte nativo a temas (via shadcn/ui).
- **Type Safe Routing**: Navegação 100% tipada com TanStack Router.
- **Componentes Acessíveis**: Baseado em Radix UI para máxima acessibilidade.

---

## 🏗️ Estrutura de Pastas
- `src/app`: Definição de rotas e layouts estruturais.
- `src/components`: Primitivas de UI e componentes de negócio.
- `src/hooks`: Lógica de estado e queries reutilizáveis.
- `src/stores`: Estados globais (Auth, Preferências).

---

## 🛠️ Guia de Inicialização

1. **Configuração**:
   ```bash
   cp .env.example .env
   # Ajuste VITE_API_URL
   ```

2. **Execução**:
   ```bash
   npm install
   npm run dev
   ```

---

## 🔐 Perfis Suportados
- **Técnico**: Operações de sala.
- **Gestor**: Visão tática da unidade.
- **Admin**: Controle total do ecossistema.
